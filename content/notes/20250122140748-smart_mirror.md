## Software {#software}


### MagicMirror {#magicmirror}

<https://magicmirror.builders/>
The open source modular smart mirror platform.


#### Notes {#notes}

Well known Open Source project, with a lot of development over the years
Build with Node.js, client on Electron (without much work it could be changed to run on a browser)
Extensive list of plugins
The code is quite large, and it may be hard to adapt


### ReflectOS {#reflectos}

<https://reflect-os.github.io/>
The approachable, configurable, and extensible Open Source Operating System for your Raspberry Pi smart mirror project.


#### Notes {#notes}

Very similar in spirit to MagicMirror, but it's a whole reimplementation written in Elixir
It's designed to run embedded with a Linux distro designed to only run Elixir without any other bloat
It has a dedicated app to control the mirror from the phone


### Senses Smart Mirror {#senses-smart-mirror}

<https://senses-smartmirror.com/>
Is modern, clean &amp; minimalistic design software for your Smart Mirror. Senses is open source and rich of widgets ready to provide you an amazing view of day-to-day data.


#### Notes {#notes}

No commits since January 2024
Build with Vue.js and Typescript, client on Electron
It also has a dedicated app to control widgets


### Evancohen smart-mirror {#evancohen-smart-mirror}

<https://github.com/evancohen/smart-mirror>
A DIY voice controlled smart mirror with IoT integration.


#### Notes {#notes}

It's been a dead project since 2021


### Clancr mirr.OS {#clancr-mirr-dot-os}

<https://glancr.net/mirr-os/>
mirr.OS is designed to provide you with an operating system that you can use for your Smart Mirror, even if you are a technical layman.


#### Notes {#notes}

It has licenses for commercial use
There seems to be [open source code](https://gitlab.com/glancr)
The project looks a bit in the air
