My use of Thunderbird is pretty basic, I don't make use of the calendar, chat or task manager (I cover some of those with emacs)
My current use case for it is to unify all my emails in a single client, which it does really well.


## Useful extensions {#useful-extensions}


### Markdown Here Revival {#markdown-here-revival}

source: <https://gitlab.com/jfx2006/markdown-here-revival>
This extension is almost essential for anyone that knows a bit of Markdown.
It adds a side panel with a preview for markdown when you write an email, and it automatically translates your markdow on send so that the receiver sees the proper format.
This is specially useful for whenever you need to send or explain some codeblocks, it even adds syntax highlighting.
![](/ox-hugo/thunderbird_markdown.png)


### Send Later {#send-later}

source: <https://gitlab.com/jfx2006/markdown-here-revival>
This adds a send later button that allows for scheduling messages, specially useful when you want to write a message on a weekend but send it on monday, or send it in the morning so that your colleagues don't see you writing emails at 2am
