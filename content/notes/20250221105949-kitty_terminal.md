## SSH {#ssh}

When connecting through `ssh` to some machines like a [Raspberry Pi]({{< relref "20250123132522-raspberry_pi.md" >}}), some TUI programs like `htop` or [tmux]({{< relref "20250221110247-tmux.md" >}}) can fail to run due to the `$TERM` variable being `xterm-kitty` and the machine may not have the terminfo definitions for it.
To solve this, Kitty has a built-in SSH feature:

```sh
kitty +kitten ssh user@host
```
