## Glow effect {#glow-effect}

To make a glow effect on any object you just need to duplicate it and apply blur to the clone.
This also gives a lot of control on the ammount of glow and the color.


## Wrap text around a circle {#wrap-text-around-a-circle}

Select both the text and the circle and navigate to **Text &gt; Put on Path**

All modifications to location, rotation and curvature must now be done to the circle.
To wrap the text around the inside instead, you just need to reverse the path of the circle before the first step, you can do so with **Path &gt; Reverse**. (remember to make the circle a path first)


## Guides {#guides}

Guides can be incredibly useful to have as reference in any kind of composition, avoiding relying on the automatic snapping.


### Create a guide {#create-a-guide}

To create a guide you can just click and drag from either the vertical or horizontal ruler on the edges of the canvas

You can also create guides from an object, like a geometric shape using **Objects &gt; Objects to guides** (Shift + G)


### Edit a guide {#edit-a-guide}

You can open the edit menu of any guide by double clicking on it.

Alternatively you can also drag a guide with the mouse and change its angle while pressing shift

## Issues and workarounds {#issues-and-workarounds}


### GTK hangs when rearranging layers {#gtk-hangs-when-rearranging-layers}

When I drag and drop a layer using the **Layers and Objects** menu GTK hangs and doesn't allow me to use any mouse input for up to a couple of minutes.
The simple workaround is to not use the drag and drop feature and rearrange layers with the arrows in the previously mentioned menu.


### Pasting emojis can result on crash {#pasting-emojis-can-result-on-crash}

Use [emojipedia.org](https://emojipedia.org) to download images of the emojis, that way font is not a factor
