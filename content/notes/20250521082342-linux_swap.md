## Steps to create a swapfile {#steps-to-create-a-swapfile}


### Create the swapfile using `dd` {#create-the-swapfile-using-dd}

Use the `dd` command to create a swapfile of the desired size. For example:

```bash
sudo dd if=/dev/zero of=/swapfile bs=1M count=1024
```

Explanation:

-   `if=/dev/zero`: Input file is a stream of zero bytes.
-   `of=/swapfile`: Output file is the swapfile to be created.
-   `bs=1M`: Block size is 1 megabyte.
-   `count=1024`: Number of blocks to write (1024 blocks of 1 MB = 1 GB).


### Set the correct permissions {#set-the-correct-permissions}

Restrict access to the swapfile for security:

```bash
sudo chmod 600 /swapfile
```


### Format the file as swap {#format-the-file-as-swap}

Prepare the file to be used as swap space:

```bash
sudo mkswap /swapfile
```


### Enable the swapfile {#enable-the-swapfile}

Activate the swapfile:

```bash
sudo swapon /swapfile
```


### Make the swapfile permanent {#make-the-swapfile-permanent}

Add the swapfile entry to the `fstab` file:

```bash
echo '/swapfile none swap sw 0 0' | sudo tee -a /etc/fstab
```

> [!warning] Make sure to check this line wan't there already, or you are mounting a swap partition
